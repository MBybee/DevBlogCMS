How blogging should be


Some blog engines are super fancy. Some are filled with bugs and complicated issues. Some are really hard to use

DevBlogCMS is a fully open tool that allows you to have complete control, and a backend done in fairly simple Perl. Install a few Perl modules on your desktop, set things up just the way you want them, and then blog. When you're done, just upload your new entries to your site. There's nothing to it. There's no hacking exposure because there's no database, no scripts. It's just static HTML and CSS files. There isn't even any javascript.
